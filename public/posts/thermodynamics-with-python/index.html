Thermodynamics is one of the most fundamental branches of physics, governing everything from steam engines to biological processes. While the mathematical equations can seem abstract, Python provides us with powerful tools to visualize and understand these concepts through computational examples. In this post, we’ll explore the key principles of thermodynamics and bring them to life with interactive Python code.

What is Thermodynamics?#

Thermodynamics is the study of heat, energy, and their transformations. It’s built on four fundamental laws that describe how energy behaves in physical systems. These laws govern everything from the efficiency of car engines to the metabolism in our cells.

Let’s start by setting up our Python environment with the necessary libraries:

The Ideal Gas Law: Foundation of Thermodynamics#

The ideal gas law is perhaps the most important equation in thermodynamics: PV = nRT

Where:

  • P = Pressure (Pa)
  • V = Volume (m³)
  • n = Number of moles
  • R = Gas constant (8.314 J/mol·K)
  • T = Temperature (K)

Let’s create a Python class to model an ideal gas:

class IdealGas:
    def __init__(self, n_moles=1.0):
        self.n = n_moles  # moles
        self.R = constants.R  # Gas constant
    
    def pressure(self, volume, temperature):
        """Calculate pressure given volume and temperature"""
        return (self.n * self.R * temperature) / volume
    
    def volume(self, pressure, temperature):
        """Calculate volume given pressure and temperature"""
        return (self.n * self.R * temperature) / pressure
    
    def temperature(self, pressure, volume):
        """Calculate temperature given pressure and volume"""
        return (pressure * volume) / (self.n * self.R)

# Example: 1 mole of gas
gas = IdealGas(n_moles=1.0)

# Calculate pressure at different temperatures
temperatures = np.linspace(200, 400, 100)  # K
volume = 0.024  # m³ (24 liters)
pressures = [gas.pressure(volume, T) for T in temperatures]

# Plot P vs T relationship
plt.figure(figsize=(10, 6))
plt.plot(temperatures, np.array(pressures)/1000, 'b-', linewidth=2)
plt.xlabel('Temperature (K)')
plt.ylabel('Pressure (kPa)')
plt.title('Ideal Gas: Pressure vs Temperature (Constant Volume)')
plt.grid(True, alpha=0.3)
plt.show()

The First Law of Thermodynamics: Energy Conservation#

The first law states that energy cannot be created or destroyed, only transformed: ΔU = Q - W

Where:

  • ΔU = Change in internal energy
  • Q = Heat added to the system
  • W = Work done by the system

Let’s simulate different thermodynamic processes:

class ThermodynamicProcess:
    def __init__(self, gas, initial_state):
        self.gas = gas
        self.P1, self.V1, self.T1 = initial_state
        
    def isothermal_expansion(self, final_volume):
        """Isothermal process: constant temperature"""
        T2 = self.T1  # Temperature constant
        P2 = self.gas.pressure(final_volume, T2)
        
        # Work done in isothermal process
        work = self.gas.n * self.gas.R * T2 * np.log(final_volume / self.V1)
        
        # For isothermal process: ΔU = 0, so Q = W
        heat = work
        delta_u = 0
        
        return {
            'final_state': (P2, final_volume, T2),
            'work': work,
            'heat': heat,
            'delta_u': delta_u,
            'process_type': 'isothermal'
        }
    
    def adiabatic_expansion(self, final_volume, gamma=1.4):
        """Adiabatic process: no heat exchange (Q = 0)"""
        # For adiabatic process: PV^γ = constant
        P2 = self.P1 * (self.V1 / final_volume) ** gamma
        T2 = self.T1 * (self.V1 / final_volume) ** (gamma - 1)
        
        # Work done in adiabatic process
        work = (self.P1 * self.V1 - P2 * final_volume) / (gamma - 1)
        
        # For adiabatic process: Q = 0, so ΔU = -W
        heat = 0
        delta_u = -work
        
        return {
            'final_state': (P2, final_volume, T2),
            'work': work,
            'heat': heat,
            'delta_u': delta_u,
            'process_type': 'adiabatic'
        }

# Example: Compare isothermal vs adiabatic expansion
gas = IdealGas(n_moles=1.0)
initial_state = (101325, 0.024, 300)  # 1 atm, 24L, 300K

process = ThermodynamicProcess(gas, initial_state)

# Expand to double the volume
final_volume = 0.048  # 48L

isothermal = process.isothermal_expansion(final_volume)
adiabatic = process.adiabatic_expansion(final_volume)

print("Isothermal Expansion:")
print(f"Work done: {isothermal['work']:.2f} J")
print(f"Heat added: {isothermal['heat']:.2f} J")
print(f"Final temperature: {isothermal['final_state'][2]:.2f} K")

print("\nAdiabatic Expansion:")
print(f"Work done: {adiabatic['work']:.2f} J")
print(f"Heat added: {adiabatic['heat']:.2f} J")
print(f"Final temperature: {adiabatic['final_state'][2]:.2f} K")

Visualizing Thermodynamic Cycles: The Carnot Engine#

The Carnot cycle represents the most efficient heat engine possible. Let’s simulate and visualize this cycle:

def carnot_cycle(T_hot, T_cold, V1, V2, V3, V4, n_moles=1.0):
    """
    Simulate a Carnot cycle with four processes:
    1-2: Isothermal expansion at T_hot
    2-3: Adiabatic expansion
    3-4: Isothermal compression at T_cold
    4-1: Adiabatic compression
    """
    gas = IdealGas(n_moles)
    gamma = 1.4  # For diatomic gas
    
    # State 1: Initial state
    P1 = gas.pressure(V1, T_hot)
    
    # Process 1-2: Isothermal expansion at T_hot
    P2 = gas.pressure(V2, T_hot)
    Q12 = n_moles * constants.R * T_hot * np.log(V2/V1)
    W12 = Q12  # ΔU = 0 for isothermal
    
    # Process 2-3: Adiabatic expansion
    P3 = P2 * (V2/V3)**gamma
    T3 = T_cold  # Final temperature after adiabatic expansion
    Q23 = 0  # Adiabatic process
    W23 = (P2*V2 - P3*V3)/(gamma-1)
    
    # Process 3-4: Isothermal compression at T_cold
    P4 = gas.pressure(V4, T_cold)
    Q34 = n_moles * constants.R * T_cold * np.log(V4/V3)
    W34 = Q34  # ΔU = 0 for isothermal
    
    # Process 4-1: Adiabatic compression
    Q41 = 0  # Adiabatic process
    W41 = (P4*V4 - P1*V1)/(gamma-1)
    
    # Total work and heat
    W_total = W12 + W23 + W34 + W41
    Q_in = Q12  # Heat input from hot reservoir
    Q_out = -Q34  # Heat output to cold reservoir
    
    # Efficiency
    efficiency = W_total / Q_in
    carnot_efficiency = 1 - T_cold/T_hot
    
    return {
        'states': [(P1, V1, T_hot), (P2, V2, T_hot), (P3, V3, T_cold), (P4, V4, T_cold)],
        'work_total': W_total,
        'heat_in': Q_in,
        'heat_out': Q_out,
        'efficiency': efficiency,
        'carnot_efficiency': carnot_efficiency
    }

# Simulate Carnot cycle
T_hot = 600  # K
T_cold = 300  # K
V1, V2, V3, V4 = 0.01, 0.02, 0.04, 0.02  # m³

cycle = carnot_cycle(T_hot, T_cold, V1, V2, V3, V4)

# Plot P-V diagram
volumes = [V1, V2, V3, V4, V1]  # Close the cycle
pressures = [state[0] for state in cycle['states']] + [cycle['states'][0][0]]

plt.figure(figsize=(10, 8))
plt.plot(np.array(volumes)*1000, np.array(pressures)/1000, 'bo-', linewidth=2, markersize=8)
plt.xlabel('Volume (L)')
plt.ylabel('Pressure (kPa)')
plt.title('Carnot Cycle P-V Diagram')
plt.grid(True, alpha=0.3)

# Add process labels
labels = ['1→2 (Isothermal)', '2→3 (Adiabatic)', '3→4 (Isothermal)', '4→1 (Adiabatic)']
for i, label in enumerate(labels):
    plt.annotate(label, xy=(volumes[i]*1000, pressures[i]/1000), 
                xytext=(10, 10), textcoords='offset points')

plt.show()

print(f"Carnot Engine Results:")
print(f"Work output: {cycle['work_total']:.2f} J")
print(f"Heat input: {cycle['heat_in']:.2f} J")
print(f"Actual efficiency: {cycle['efficiency']:.3f}")
print(f"Theoretical Carnot efficiency: {cycle['carnot_efficiency']:.3f}")

The Second Law: Entropy and the Arrow of Time#

The second law of thermodynamics introduces the concept of entropy - a measure of disorder that always increases in isolated systems. Let’s explore this with a mixing simulation:

def entropy_mixing_simulation(n_particles=1000, n_steps=1000):
    """
    Simulate the mixing of two gases and calculate entropy change
    """
    # Initialize particles: half on left, half on right
    positions = np.zeros(n_particles)
    positions[:n_particles//2] = 0  # Left side
    positions[n_particles//2:] = 1  # Right side
    
    # Track the number of particles on each side over time
    left_counts = []
    entropies = []
    
    for step in range(n_steps):
        # Random walk: each particle has 50% chance to move
        moves = np.random.random(n_particles) < 0.1  # 10% chance to move each step
        positions[moves] = 1 - positions[moves]  # Flip side
        
        # Count particles on left side
        left_count = np.sum(positions == 0)
        left_counts.append(left_count)
        
        # Calculate mixing entropy (simplified)
        if left_count > 0 and left_count < n_particles:
            p_left = left_count / n_particles
            p_right = 1 - p_left
            entropy = -(p_left * np.log(p_left) + p_right * np.log(p_right))
        else:
            entropy = 0
        entropies.append(entropy)
    
    return left_counts, entropies

# Run simulation
left_counts, entropies = entropy_mixing_simulation()

# Plot results
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(12, 10))

# Particle distribution over time
ax1.plot(left_counts, 'b-', alpha=0.7)
ax1.axhline(y=500, color='r', linestyle='--', label='Equilibrium (50%)')
ax1.set_xlabel('Time Steps')
ax1.set_ylabel('Particles on Left Side')
ax1.set_title('Gas Mixing: Particle Distribution Over Time')
ax1.legend()
ax1.grid(True, alpha=0.3)

# Entropy over time
ax2.plot(entropies, 'g-', linewidth=2)
ax2.set_xlabel('Time Steps')
ax2.set_ylabel('Mixing Entropy')
ax2.set_title('Entropy Increase During Mixing')
ax2.grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

print(f"Initial entropy: {entropies[0]:.4f}")
print(f"Final entropy: {entropies[-1]:.4f}")
print(f"Entropy increase: {entropies[-1] - entropies[0]:.4f}")

Real-World Application: Heat Engine Efficiency#

Let’s create a practical example by modeling a car engine and calculating its efficiency:

class HeatEngine:
    def __init__(self, T_hot, T_cold):
        self.T_hot = T_hot  # Hot reservoir temperature (K)
        self.T_cold = T_cold  # Cold reservoir temperature (K)
        self.carnot_efficiency = 1 - T_cold/T_hot
    
    def otto_cycle_efficiency(self, compression_ratio, gamma=1.4):
        """Calculate Otto cycle (car engine) efficiency"""
        return 1 - (1/compression_ratio)**(gamma-1)
    
    def diesel_cycle_efficiency(self, compression_ratio, cutoff_ratio, gamma=1.4):
        """Calculate Diesel cycle efficiency"""
        r = compression_ratio
        rc = cutoff_ratio
        return 1 - (1/gamma) * (1/r**(gamma-1)) * ((rc**gamma - 1)/(rc - 1))

# Compare different engine types
T_combustion = 2000  # K (combustion temperature)
T_exhaust = 800     # K (exhaust temperature)

engine = HeatEngine(T_combustion, T_exhaust)

# Calculate efficiencies for different compression ratios
compression_ratios = np.linspace(6, 12, 50)
otto_efficiencies = [engine.otto_cycle_efficiency(r) for r in compression_ratios]
diesel_efficiencies = [engine.diesel_cycle_efficiency(r, 2.0) for r in compression_ratios]

plt.figure(figsize=(12, 8))
plt.plot(compression_ratios, np.array(otto_efficiencies)*100, 'b-', 
         linewidth=2, label='Otto Cycle (Gasoline Engine)')
plt.plot(compression_ratios, np.array(diesel_efficiencies)*100, 'r-', 
         linewidth=2, label='Diesel Cycle')
plt.axhline(y=engine.carnot_efficiency*100, color='g', linestyle='--', 
           linewidth=2, label=f'Carnot Limit ({engine.carnot_efficiency*100:.1f}%)')

plt.xlabel('Compression Ratio')
plt.ylabel('Efficiency (%)')
plt.title('Heat Engine Efficiency vs Compression Ratio')
plt.legend()
plt.grid(True, alpha=0.3)
plt.ylim(0, 70)
plt.show()

print(f"Carnot efficiency limit: {engine.carnot_efficiency*100:.1f}%")
print(f"Typical car engine efficiency (r=10): {engine.otto_cycle_efficiency(10)*100:.1f}%")
print(f"Typical diesel engine efficiency (r=18): {engine.diesel_cycle_efficiency(18, 2.0)*100:.1f}%")

Interactive Thermodynamics Dashboard#

Finally, let’s create an interactive visualization that lets us explore how different parameters affect thermodynamic processes:

def create_thermodynamics_dashboard():
    """Create an interactive dashboard for exploring thermodynamic relationships"""
    
    # Create sample data for different processes
    temperatures = np.linspace(200, 600, 100)
    volumes = np.linspace(0.01, 0.1, 100)
    
    # Create a grid for 3D plotting
    T_grid, V_grid = np.meshgrid(temperatures, volumes)
    
    # Calculate pressure surface for ideal gas (n=1 mol)
    gas = IdealGas(n_moles=1.0)
    P_grid = np.zeros_like(T_grid)
    for i in range(len(temperatures)):
        for j in range(len(volumes)):
            P_grid[j, i] = gas.pressure(volumes[j], temperatures[i])
    
    # Create 3D surface plot
    fig = plt.figure(figsize=(15, 5))
    
    # 3D surface plot
    ax1 = fig.add_subplot(131, projection='3d')
    surf = ax1.plot_surface(T_grid, V_grid*1000, P_grid/1000, 
                           cmap='viridis', alpha=0.8)
    ax1.set_xlabel('Temperature (K)')
    ax1.set_ylabel('Volume (L)')
    ax1.set_zlabel('Pressure (kPa)')
    ax1.set_title('Ideal Gas: P-V-T Surface')
    
    # Contour plot
    ax2 = fig.add_subplot(132)
    contour = ax2.contour(T_grid, V_grid*1000, P_grid/1000, levels=20)
    ax2.clabel(contour, inline=True, fontsize=8)
    ax2.set_xlabel('Temperature (K)')
    ax2.set_ylabel('Volume (L)')
    ax2.set_title('Pressure Contours (kPa)')
    
    # Efficiency comparison
    ax3 = fig.add_subplot(133)
    T_hot_range = np.linspace(400, 1000, 100)
    T_cold = 300
    carnot_eff = 1 - T_cold/T_hot_range
    real_eff = carnot_eff * 0.6  # Assume 60% of Carnot efficiency
    
    ax3.plot(T_hot_range, carnot_eff*100, 'r-', linewidth=2, label='Carnot Limit')
    ax3.plot(T_hot_range, real_eff*100, 'b-', linewidth=2, label='Real Engine')
    ax3.fill_between(T_hot_range, real_eff*100, carnot_eff*100, alpha=0.3, color='gray')
    ax3.set_xlabel('Hot Reservoir Temperature (K)')
    ax3.set_ylabel('Efficiency (%)')
    ax3.set_title('Engine Efficiency vs Temperature')
    ax3.legend()
    ax3.grid(True, alpha=0.3)
    
    plt.tight_layout()
    plt.show()

create_thermodynamics_dashboard()

Conclusion#

Through these Python examples, we’ve explored the fundamental concepts of thermodynamics:

  1. The Ideal Gas Law provides the foundation for understanding pressure-volume-temperature relationships
  2. The First Law shows us how energy is conserved in all processes
  3. The Second Law introduces entropy and explains why certain processes are irreversible
  4. Heat Engines demonstrate practical applications and efficiency limits

Python’s computational power allows us to visualize these abstract concepts and explore “what-if” scenarios that would be difficult to investigate experimentally. Whether you’re studying physics, engineering, or just curious about how the world works, computational thermodynamics provides powerful insights into the fundamental laws that govern energy and heat.

The code examples in this post can be extended and modified to explore other thermodynamic phenomena like phase transitions, chemical reactions, or more complex engine cycles. The beauty of computational physics is that it makes the invisible visible and the complex understandable.
